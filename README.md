Encodeur du JTX
===============

##Instructions

* Placer `JTX-Encoder.exe` dans le dossier contenant les vidéos à encoder et l'exécuter
* Suivre les instructions à l'écran et profiter !

##Remarques

* Pour que le programme marche correctement, il faut que le dossier en cours donne les drots d'exécution par défaut aux nouveaux fichiers.
